# English

### How to suggest an improvement

- Create a GitHub account if you don't have one
- Click “Discussions” on the GitHub repository
- Create a new discussion and post your idea

# Français

### Comment suggérer une amélioration

1. Créez un compte GitHub si vous n'en avez pas
2. Cliquez sur "Discussions" du dépôt GitHub
3. Créez une nouvelle discussion et écrivez votre idée
